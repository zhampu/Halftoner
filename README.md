# Halftoner

## Instructions

Drag n' Drop the image to the Processing canvas and play with it

 - Arrow UP & Down = Dot Size
 - Arrow Left & Right = Steps between dots
 - + = zoom +
 - - = zoom - 
 - e = export pdf
 - c = cyan
 - m = magenta
 - y = yellow
 - k = black
 
 
 ## NOTES
 
 Do not go crazy on the amount on the steps or the program will crash
